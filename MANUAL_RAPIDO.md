# Manual Rapido - MediaLayers

## Operacao em 2 minutos

1. Abra o MediaLayers.
2. Arraste uma imagem, video ou audio para o painel Entradas.
3. Clique em Cue para mandar a layer ao Preview.
4. Clique em Take para mandar a layer ao Program.
5. Confira a janela de saida.

## Controle remoto no celular

1. Abra a aba Controle Remoto.
2. Copie um dos links exibidos.
3. Abra esse link no celular na mesma rede.
4. Use Anterior, Proximo, TAKE e CLEAR.

## Browser Source

1. Clique em + Browser.
2. Informe a URL.
3. Ajuste a layer nas Propriedades.
4. Use Cue e depois Take.

## NDI

1. Clique em + NDI.
2. Escolha a fonte encontrada.
3. Valide no Preview.
4. Use Iniciar NDI Output quando quiser publicar a saida.

## Atualizacao automatica

1. Abra a aba Atualizacoes.
2. Informe a URL onde voce publicou a release.
3. Clique em Salvar.
4. Clique em Verificar agora.
5. Quando o download terminar, clique em Instalar update.

## Se algo der errado

1. Use Restaurar ou Reset no layout.
2. Verifique se o celular esta na mesma rede.
3. Confirme se latest.yml esta publicado para o auto-update.
4. Teste sem NDI para isolar problemas de rede.
